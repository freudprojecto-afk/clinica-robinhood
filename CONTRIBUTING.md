# Guia de Contribuição

## Configuração do Ambiente

1. Instale as dependências:
```bash
npm install
```

2. Configure as variáveis de ambiente (veja README.md)

3. Execute o projeto em desenvolvimento:
```bash
npm run dev
```

## Estrutura de Branches

- `main` - Branch principal (produção)
- `develop` - Branch de desenvolvimento
- `feature/*` - Novas funcionalidades
- `fix/*` - Correções de bugs

## Padrões de Código

- Use TypeScript para type safety
- Siga os padrões do ESLint configurado
- Componentes devem ser funcionais e usar hooks
- Use Tailwind CSS para estilização
- Mantenha o design consistente com o tema Robinhood

## Commits

Use mensagens descritivas:
- `feat: adiciona nova funcionalidade`
- `fix: corrige bug em X`
- `style: ajusta estilos`
- `refactor: refatora código`
